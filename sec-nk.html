    Section 1.4 Nick’s Section: Limits of Trigonometric and Exponential Functions

    Subsection Continuity of Trigonometric and Exponential Functions

    It can be shown that the trigonometric and exponential functions such as \(\sin x\text{,}\) \(\cos x\text{,}\) \(\tan x\text{,}\) \(e^x\text{,}\) etc. are continuous at every input value where they are defined. As a consequence of this, many limits of this sort may be evalauted by simply “plugging in” the limiting value.

    Example 1.4.1.

    \begin{equation*} \lim_{x\to \pi/3}\tan x=\tan(\pi/3)=\sqrt{3} \end{equation*}

    Example 1.4.2.

    \begin{equation*} \lim_{x\to \pi/2}\frac{\sin x}{x}=\frac{\sin(\pi/2)}{\pi/2}=\frac{1}{\pi/2}=\frac{2}{\pi} \end{equation*}

    Subsection The Squeeze Theorem

    The more difficult question is how to evaluate limits of these sorts of functions which are in the indeterminate form \(0/0\text{.}\) One valuable tool to help with limits of these sorts is the Squeeze Theorem.

    Theorem 1.4.3. The Squeeze Theorem.

    Let \(I\) be an open interval containing the number \(c\text{.}\) If \(f\text{,}\) \(g\text{,}\) and \(h\) are function such that
    \begin{equation*} g(x)\leq f(x)\leq h(x) \end{equation*}
    for all values of \(x\) in the interval \(I\) (except possibly for \(c\)) and
    \begin{equation*} \lim_{x\to c}g(x)=\lim_{x\to c}h(x)=L, \end{equation*}
    then \(\lim_{x\to c}f(x)=L\) as well.

    Example 1.4.4.

    Suppose that \(f\) is a function such that
    \begin{equation*} \overbrace{1+2x-x^2}^{g(x)}\leq f(x)\leq \overbrace{x^2-2x+3}^{h(x)} \end{equation*}
    for all \(x\text{.}\) Since
    \begin{equation*} \lim_{x\to 1}(1+2x-x^2)=2\quad\text{and}\quad\lim_{x\to 1}(x^2-2x+3)=2\text{,} \end{equation*}
    we may conclude that \(\lim_{x\to 1}f(x)=2\) by virtue of the Squeeze Theorem.

    Checkpoint 1.4.5.

    Suppose that \(f\) is a function such that
    \begin{equation*} x+1\leq f(x)\leq e^x \end{equation*}
    for all \(x\text{.}\) Can the Squeeze Theorem be used to find the value of \(\lim_{x\to 1}f(x)\text{?}\)
    • Yes, \(\lim_{x\to 1}f(x)=2\text{.}\)
    • Yes, \(\lim_{x\to 1}f(x)=e^2\text{.}\)
    • No, it is not possible to determine the value of \(\lim_{x\to 1}f(x)\text{.}\)
    • We cannot use the Squeeze Theroem to determine the value of \(\lim_{x\to 1}f(x)\) since \(\lim_{x\to 1}x+1\neq \lim_{x\to 1}e^x\text{.}\)

    Subsection Two Important Limits

    It can be shown by a variety of arguments that \(\cos\theta\leq \frac{\sin\theta}{\theta}\leq \frac{1}{\cos\theta}\) for values of \(\theta\) near \(0\text{.}\) Once this is established, the Squeeze Theorem then permits the following conclusions

    Corollary 1.4.6.

    \begin{equation*} \lim_{\theta\to 0}\frac{\sin(\theta)}{\theta}=1\quad\text{and}\quad \lim_{\theta\to 0}\frac{1-\cos(\theta)}{\theta}=0 \end{equation*}
    Once we know these two trigonometric limits, we can use them along with limit properties in a variety of ways illustrated in the following examples.

    Example 1.4.7.

    Evaluate \(\lim_{x\to 0}\frac{1-\cos x}{2x}\text{.}\)
    Solution.
    By factoring out the constant \(1/2\text{,}\)
    \begin{equation*} \lim_{x\to 0}\frac{1-\cos x}{2x} = \frac{1}{2}\overbrace{\lim_{x\to 0}\frac{1-\cos x}{x}}^{0} =\frac{1}{2}(0)=0 \end{equation*}

    Example 1.4.8.

    Evaluate \(\lim_{x\to 0}\frac{7\sin(x)}{6x}\text{.}\)
    Solution.
    By factoring out the constant \(7/6\text{,}\)
    \begin{equation*} \lim_{x\to 0}\frac{7\sin(x)}{6x} = \frac{7}{6}\overbrace{\lim_{x\to 0}\frac{\sin(x)}{x}}^{1} =\frac{7}{6}(1)=1 \end{equation*}

    Example 1.4.9.

    Evaluate \(\lim_{x\to 0}\frac{\sin(8x)}{x}\text{.}\)
    Solution.
    Multiplying by the number \(1\) will not change the value of the limit. Here, we choose the form \(\frac{8}{8}\text{.}\) With this,
    \begin{equation*} \lim_{x\to 0}\frac{\sin(8x)}{x} = \lim_{x\to 0}\frac{8\sin(8x)}{8x} = 8\lim_{x\to 0}\frac{\sin(8x)}{8x} \end{equation*}
    This final limit is of the form \(\lim_{\theta\to 0}\frac{\sin(\theta)}{\theta}\) by taking \(\theta=8x\text{,}\) and hence
    \begin{equation*} \lim_{x\to 0}\frac{\sin(8x)}{x}=\cdots=8\overbrace{\lim_{x\to 0}\frac{\sin(8x)}{8x}}^{1}=8\text{.} \end{equation*}

    Exercises Exercises

    1.

    \(\lim_{x\to 0}\left(3x+4\cos(2x+\pi)\right)\)
    Answer.
    \(-4\)

    2.

    On what intervals is \(f(x)=\ln(2x-3)\) continuous?
    Answer.
    \((3/2,\infty)\)

    3.

    \(\lim_{x\to 0}\frac{2x-5\sin(x)}{x}\)
    Answer.
    -13

    4.

    \(\lim_{x\to 0}\frac{\tan(x)}{x}\)
    Answer.
    1

    5.

    \(\lim_{x\to 0}\frac{\sin(2x)-x\cos(3x)}{4x}\)
    Answer.
    1/4
